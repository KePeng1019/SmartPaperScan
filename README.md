# SmartPaperScan
this is an android project write in kotlin which detect paper or rectangle by opencv, you can take a picture and crop it(使用开源opencv动态检测桌面上预览的文档，确认预览后进行拍摄剪裁，矫正拍摄出的照片，项目的实现是用kotlin写的)
